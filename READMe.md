# Route Tracking Map using Google Maps API

## Project Description
This project is a simple web application that displays a Google Map and allows users to calculate a route from their current location to a selected destination. The destination is selected dynamically by clicking on the map.

## Features
- Displays a Google Map
- Detects the user's current location
- Allows users to click anywhere on the map to set a destination
- Calculates the driving route from the current location to the selected destination

## Requirements
- Internet connection
- Web browser (Chrome, Edge, Firefox, etc.)
- Google Maps API Key
- XAMPP (for running on localhost)

## How to Run the Project
1. Download or clone the project files.
2. Place the project folder inside the **htdocs** directory of XAMPP.
3. Start **Apache** in the XAMPP Control Panel.
4. Open a browser and go to:

http://localhost/google-maps-route-tracking

5. Allow location access when prompted.
6. Click anywhere on the map to choose a destination.
7. Click **Get Route** to display the route.

## Project Files
- `index.html` – Main webpage
- `style.css` – User interface design
- `script.js` – Google Maps and route calculation logic
